Security Features Implemented:

  • CSRF Protection: Form includes CSRF token to prevent Cross-Site Request Forgery
  • XSS Prevention: Input sanitization removes HTML tags and escapes special characters
  • Input Validation: Name field only allows letters, spaces, and common punctuation
  • SQL Injection Prevention: Using Django ORM with parameterized queries
  • Output Escaping: All user inputs are escaped when displayed
